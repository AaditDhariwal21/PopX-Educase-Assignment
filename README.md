# PopX Frontend

A pixel-perfect mobile app interface built with React and Tailwind CSS, based on an Adobe design mockup.
This is and internsip assignment for Educase

## Tech Stack

- React 19
- Tailwind CSS 4
- React Router DOM
- Vite

## Pages

| # | Page             | URL         | Description                                      |
|---|------------------|-------------|--------------------------------------------------|
| 1 | Landing Page     | `/`         | Welcome screen with Create Account and Login buttons |
| 2 | Login Page       | `/login`    | Sign in form with email and password fields      |
| 3 | Register Page    | `/register` | Account creation form with 5 fields and agency selection |
| 4 | Account Settings | `/account`  | User profile page with avatar, name, and email   |

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Deployment

The project is hosted on Vercel.
